# Project Status

## Fork Information

**This is an unofficial fork** of the HBDI PII Toolkit (pbD-Toolkit). This fork is not maintained by or affiliated with the Hessian Commissioner for Data Protection and Freedom of Information (HBDI).

## Original Project

- **Original Name**: pbD-Toolkit (Personenbezogene Daten Toolkit)
- **Original Maintainer**: Hessian Commissioner for Data Protection and Freedom of Information (HBDI)
- **Original Repository**: [hessen-datenschutz/pbd-toolkit](https://github.com/hessen-datenschutz/pbd-toolkit)
- **Original Contact**: Department 3.2 - Technical Data Protection Audits
- **Original Website**: [datenschutz.hessen.de](https://datenschutz.hessen.de)

## Fork Status

### Current Version

This fork includes various improvements and modifications over the original project:

- Enhanced file format support
- Multiple output formats (CSV, JSON, XLSX)
- Improved CLI options
- Better error handling
- Performance optimizations
- Comprehensive documentation

### Differences from Original

This fork may differ from the original in:
- Additional features
- Bug fixes
- Code structure
- Documentation
- Dependencies

**Important**: This fork is not guaranteed to be compatible with the original project.
